Príklad 1
Nájdite lokálne extrémy funkcie f(x,y) = x^3 + 3xy^2 - 51x - 24y.

Pre navigáciu medzi slajdami použite šípky.

f(x, y) = x^3 + 3xy^2 - 51x - 24y

Majme funkciu f(x, y) = x^3 + 3xy^2 - 51x - 24y. Pre nájdenie lokálneho extrému funkcie je postup nasledovný:

  1. Nájsť stacionárne body,
  2. otestovať stacionárne body na možný extrém,
  3. vyšetriť body, v ktorých neexistujú parciálne derivácie podľa všetkých premenných.

f(x,y) = \color{#32BD00}{x^3} + 3\color{#32BD00}{x}y^2 - 51\color{#32BD00}{x} - 24y

\frac{\partial f(x,y)}{\partial \color{#32BD00}{x}} = 3\color{#32BD00}{x^2} + 3y^2 - 51 = 0

Začnime parciálnou deriváciou \frac{\partial f(x,y)}{\partial x}. Z tohto strašideľne vyzerajúceho zápisu môžeme vyčítať, že chceme zderivovať našu funkciu f(x,y) podľa \color{#32BD00}{x}. To znamená, že všetky \color{#803906}{y} ako keby vo funkcii neexistovali (povieme o nich, že sú to konštanty)! Poďme si to radšej zafarbiť aby sme to lepšie videli.

f(x,y) = \color{#32BD00}{x^3} + 3\color{#32BD00}{x}y^2 - \color{#32BD00}{51x} - 24y

Derivujme teda podľa \color{#32BD00}{x}:

f(x,y) = \color{#32BD00}{x^3} + \color{#32BD00}{3x}y^2 - \color{#32BD00}{51x} - 24y
\frac{\partial f(x,y)}{\partial \color{#32BD00}{x}} = \color{#32BD00}{3x^2} + \color{#32BD00}{3y^2} - \color{#32BD00}{51} = 0

f(x,y) = x^3 + \color{#803906}{3}x\color{#803906}{y^2} - 51x - \color{#803906}{24y}

\frac{\partial f(x,y)}{\partial \color{#32BD00}{x}} = \color{#32BD00}{3x^2} + \color{#32BD00}{3y^2} - \color{#32BD00}{51} = 0

\frac{\partial f(x,y)}{\partial \color{#803906}{y}} = \color{#803906}{6xy} - \color{#803906}{24} = 0

Teraz zderivujeme funkciu f(x,y) podľa \color{#803906}{y}:

f(x,y) = x^3 + \color{#803906}{3}x\color{#803906}{y^2} - 51x - \color{#803906}{24y}
\frac{\partial f(x,y)}{\partial \color{#803906}{y}} = \color{#803906}{6xy} - \color{#803906}{24} = 0

f(x,y) = x^3 + \color{#803906}{3}x\color{#803906}{y^2} - 51x - \color{#803906}{24y}

\frac{\partial f(x,y)}{\partial \color{#32BD00}{x}} = \color{#32BD00}{3x^2} + \color{#32BD00}{3y^2} - \color{#32BD00}{51} = 0

\frac{\partial f(x,y)}{\partial \color{#803906}{y}} = \color{#803906}{6xy} - \color{#803906}{24} = 0

Derivácia funkcie podľa y

f(x,y) = x^3 + 3xy^} - 51x - 24y

\frac{\partial f(x,y)}{\partial x} = 3x^2 + 3y^2 - 51 = 0

\frac{\partial f(x,y)}{\partial y} = 6xy - 24 = 0

Z druhej rovnice vyjadrime \color{#803906}{y = \frac{4}{x}}

Potrebujeme vyjadriť jednu z premenných. Z druhej rovnice to bude ľahšie, takže ideme na to:

6xy - 24 = 0     / pripočítame 24
6xy = 24     / predelíme 6-timi
xy = 4     / vyjadríme.. napríklad y
\color{#803906}{y = \frac{4}{x}}

y máme, teraz vyjadríme x z prvej rovnice...

f(x,y) = x^3 + 3xy^} - 51x - 24y

\frac{\partial f(x,y)}{\partial x} = 3x^2 + 3y^2 - 51 = 0

\frac{\partial f(x,y)}{\partial y} = 6xy - 24 = 0

Z druhej rovnice vyjadrime y = \frac{4}{x}

Z prvej rovnice dostávame:

x_1^2 = 16

x_2^2 = 1

Vyjadrujeme x z prvej rovnice:

3x^2 + 3y^2 - 51 = 0     / predelíme 3-kou
x^2 + y^2 - 17 = 0     / za y dosadíme y = \frac{4}{x}
x^2 + (\color{#803906}{\frac{4}{x}})^2 - 17 = 0
x^2 + \frac{16}{x^2} - 17 = 0     / prenásobíme s x^2 aby sme odstránili zlomky
x^\color{#803906}{4} + 16 - 17x^2 = 0
x^4 - 17x^2 + 16 = 0

Rovnicu x^4 - 17x^2 + 16 = 0 nevieme vyriešiť tak ľahko. Pomôžeme si substitúciou \color{#803906}{x^2} = \color{#D45800}{a}. Naša rovnica bude hneď vyzerať ľahšie a taktiež ju už budeme vedieť riešiť:

a^2 - 17a + 16 = 0
Kvadratická rovnica! "Veď tú poznám!"
a_1,a_2 = \frac{b \pm \sqrt{D}}{2a}
a_1,a_2 = \frac{b \pm \sqrt{(b^2 - 4ac)}}{2a}
a_1,a_2 = \frac{17 \pm \sqrt{(17^2 - 4*1*16)}}{2*1}
a_1,a_2 = \frac{17 \pm \sqrt{225}}{2}
a_1 = 16
a_2 = 1

Nesmieme zabudnúť vrátiť späť do rovnice výraz, ktorý sme substituovali. Dostávame teda:

x_1^2 = 16
x_2^2 = 1

x_1 = \sqrt{x_1^2}
x_2 = \sqrt{x_2^2}

x_1 = \sqrt{16}
x_2 = \sqrt{1}

x_1 = \pm 4
x_2 = \pm 1

x_1 a x_2 obe nadobúdajú kladné aj záporné hodnoty. To znamená, že máme do činenia so štyrma x-ovými súradnicami! Štyri x-ové súradnice = štyri body! Poďme si tieto súradnice krajšie označiť a priradiť k ním aj príslušné hodnoty:

y-ové súradnice:
y_1 = 1
y_2 = -1
y_3 = 4
y_4 = -4

Máme 4 x-ové súradnice, čiže vieme, že máme do činenia so štyrma stacionárnymi bodmi, potrebujeme k ním ešte nájsť y-ové súradnice (Stačí len dosadiť do vzorca y = \frac{4}{x}).

y_1 = \frac{4}{x_1} = \frac{4}{4} = 1
y_2 = \frac{4}{x_2} = \frac{4}{-4} = -1
y_3 = \frac{4}{x_3} = \frac{4}{1} = 4
y_4 = \frac{4}{x_4} = \frac{4}{-1} = -4

Determinant parciálnych derivácií druhého rádu v bode A:

D= \begin{bmatrix} \frac{\partial^2 f(A)}{\partial x^2} & \frac{\partial^2 f(A)}{\partial xy} \\ \frac{\partial^2 f(A)}{\partial xy} & \frac{\partial^2 f(A)}{\partial y^2} \end{bmatrix} > 0
Ak platí, potom v bode A má funkcia f(x,y) lokálny extrém.

Aby sme zistili, či sa v našich stacionárnych bodoch nachádzajú lokálne extrémy, potrebujeme pre každý stacionárny bod vypočítať determinant parciálnych derivácií druhého rádu.
